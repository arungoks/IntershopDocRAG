# Publishing Intershop Commerce Management Users

In data replication environments, organization managers can publish users from the editing system to the target system on a fine-grained base. To do so:
-
In the navigation bar, select Organization | Users.
-
Select the checkbox of the user to be published to the target system.
-
Click Publish to Live System.
This displays the list of configured target systems.

-
Select the intended target system and click Finish.
The selected users are replicated to the selected target system.
