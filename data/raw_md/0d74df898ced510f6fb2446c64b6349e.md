# Assigning Label to Multiple Products

To assign a label to multiple products at once:
-
Locate the products you want to edit.
Select the check boxes for the products on the product list view and click the

Edit Selected button. Alternatively, perform a search (see

[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html)) and click the

Edit All button to apply modifications to the search result.

-
Select the task Assign Label and click Next.
A list of existing labels is displayed.

-
Select a label from the list to assign an existing label, or provide name and ID to create a new label.
-
Click Finish.
This assigns the label to the selected products.
