# Running Scheduled Jobs Manually

-
In the Intershop System Management navigation bar, select Schedules | Scheduling.
A list of all schedules in the
*root*domain is displayed. -
Select the domain where you intend to run the schedule(s) and click Apply.
A list of all schedules in the selected domain is displayed.
- Select the checkbox(es) next to the schedule(s) you intend to run.
-
Click Run.
The scheduled job(s) is/are run immediately, regardless of any activation and run time settings.
