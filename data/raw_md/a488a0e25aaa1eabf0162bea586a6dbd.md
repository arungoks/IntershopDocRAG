# Assigning Access Privileges/Roles

To assign access privileges or roles to an existing user:
-
In the navigation bar, select Organization | Users.
-
Click the name of the user you want to edit.
This opens the user detail view.

-
Change to the Access Privileges tab or the Department Roles tab and edit the information as required.
Save the changes you make on one tab before changing to the next tab.
