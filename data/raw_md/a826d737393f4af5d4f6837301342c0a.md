# Creating Product Filters

To create a new product filter:
-
Select the channel from the context selection box, then select Catalogs | Predefined Product Filters from the navigation bar.
This displays the list of all product filters defined for this channel.

-
Click New.
This opens an empty detail view.

-
Specify the filter details as required.
Specify name, ID and description (optional), as well as the intended rules. For details about product filter rules, see

[Editing Product Filter Rules](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_editing_product_filter_rules.html).

-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
