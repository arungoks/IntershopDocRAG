# Exporting Postal Data

To export postal data in XML format:
-
In the navigation bar, select Import & Export.
The overview page is displayed.

-
In the Postal Data row, click Export.
-
Provide a name for the export file.
-
Click Export.
The export process is started immediately. The Import & Export Status dialog is displayed.

-
Download the export file.
