# Searching Administration Users

To search for system administration users:

-
In the navigation bar, select Users.
A list of all system administration users is displayed.Note:This list should at least display the default administration user, which is always created upon system initialization.
-
Specify the user's name or ID in the search field.
To perform a more detailed search based on other user information, switch to the Advanced Search by clicking Switch to Advanced Search.
-
Click Find.
A list of all users that match the query is displayed.
