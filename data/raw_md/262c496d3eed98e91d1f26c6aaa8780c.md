# Storefront Impersonation

While resolving a customer's request in a contact session (see [Starting a Contact Session](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/task_starting_a_contact_session.html)), you may need to browse the sales channel storefront on behalf of the customer. To this end, you can start an impersonation storefront session.

As the call center agent, you are automatically logged in to the storefront using the customer's credentials. This way, you are able to see the catalogs and products, prices, promotions etc. as the customer.

To start an impersonation storefront session:

-
Start (if not already running) the contact session, see
[Starting a Contact Session](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/task_starting_a_contact_session.html). -
Click Impersonate.
This opens the sales channel storefront using the customer's credentials in a new browser tab.
