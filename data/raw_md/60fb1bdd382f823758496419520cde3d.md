# Starting a Contact Session

When you select a customer, you start a contact session and can process the customer's request.

To start a contact session:

-
Search for the customer, see
[Search Customers](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/task_simple_customer_search.html). -
Select the intended customer.
This starts a contact session with the selected customer, loading the customer data into the current session context. The Intershop Customer Engagement Center now displays the customer overview page (see
[Customer Overview](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/concept_customer_overview.html)).
