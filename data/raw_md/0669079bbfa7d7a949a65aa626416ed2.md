# Exporting Short Links (CSV)

You can export existing short links in CSV format. To do so:
-
Navigate to the Import & Export page of Link Management.
-
Select the channel from the context selection box, then select Marketing | Link Management.
-
Click Import & Export.

-
In the Links (CSV) row, click Export.
The list of short links in the management context is displayed.

-
Select the short links to export and click Export.
-
Provide an export file name, then click Export again.
The export process is started immediately. The Import & Export Status dialog is displayed.

-
Download the export file.
