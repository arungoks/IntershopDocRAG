# Managing Image Settings

[Specifying the Image Upload Location for Channels](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/task_specifying_the_image_upload_location.html), it is possible to define channels that have access to image sources stored in the organization's shared file system. To adjust image settings, do the following:

-
In the navigation bar, select Master Catalogs | Image Management.
This displays the Image Upload tab.
-
Open the Image Settings tab.
This displays a list of available channels. If checked, the channel has access to image sources stored in the organization's shared file system.
- Select the channels for which you want to grant or revoke access settings.
-
Click Apply.
Note:This setting only affects image types with an empty URL prefix, see
[Creating Image Types](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/topic_managing_image_types.html#task_pim_images_types_create). Images that were assigned previously are not affected.
