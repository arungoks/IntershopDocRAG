# Accessing Receipts

-
Search for the order.
For details, see
[Searching Orders](https://knowledge.intershop.com/oh/go.php/iom/master/iom_en/topics/managing_orders/topic_searching_orders.html#task_order_search). -
In the result list, click on the order number you want to edit.
This opens the order detail page.
-
Switch to the Receipts tab.
This displays the receipt(s) for this order.
-
To view delivery details, click the icon.
This displays the delivered products and additional information.
-
To view the message history, click the icon in the Action column.
This displays the message history.
