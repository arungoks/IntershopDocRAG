Intershop's Omni-Channel Services Toolset is a collection of [framework](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163970267/framework)s that enable fast, standardized development of business web [app](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152517685/application)s like the [Intershop Contact Center](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2195271048/Intershop+Contact+Center). Based on open-source technologies, the OCST is decoupled from the [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management) core platform and uses the [REST API](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214729930/REST+framework) (as part of the [Synaptic Commerce API](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214745183/Synaptic+Commerce+API)) to access data and commerce functionality.

For details about the REST API see: [Reference - Omni-Channel Services Toolset REST API](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1788152916/Reference+-+Omni-Channel+Services+Toolset+REST+API).
