# Creating Users via Intershop Organization Management

The easiest way to create new Intershop System Management users is to log in to the Intershop Commerce Management application with the context *Operations*, create new users there and assign them the access privilege *SMC Administrator*.

For more details about how to manage administration users refer to the [Intershop Organization Management User Manual | Managing Administration Users](https://knowledge.intershop.com/oh/index.php/Display/G33630).
