# Running Product Data Feeds Manually

-
Select the channel from the context selection box, then select Catalogs | Product Data Feeds from the navigation bar.
A list of all configurations currently defined is displayed.
-
Select the configuration to use for the product data feed job and click the Run button.
This starts the product data feed process. Click the Refresh button to update the current page.

Tip:

After the job has finished, you can click the name of the result file to download it. To view the log file, click the respective entry in the column

*Last Data Feed*.
