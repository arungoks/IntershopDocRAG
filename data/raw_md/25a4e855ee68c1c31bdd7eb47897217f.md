# Searching Cashback Promotion Codes

Note:

To search for cashback promotion codes for a specific customer, perform the following steps:This feature is available for B2C channels only.

-
Select the channel from the context selection box, then select Marketing | Cashback Promotion Codes.
This opens the cashback promotion code section with the Simple Search mask.
-
Enter an e-mail address, login or customer ID into the input field.
Alternatively, click on the Select button () to open the customer search page and select a customer there.
-
Click Find or press Enter to start the search.
All available promotion codes are displayed with additional information, e.g., earned cashback, available cashback, earning order and redemption order.
-
Click the cashback promotion code name to enter the cashback promotion code search detail view.
For an explanation of the cashback promotion code search detail view, refer to
[Promotion Code Search](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/online_marketing/concept_promotion_code_search.html).
