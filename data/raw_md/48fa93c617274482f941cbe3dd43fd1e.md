# Exporting Customers

Note:

To export customers:Currently, the Order Management Tool supports CSV file export only.

-
Search for the customers.
For details, see
[Searching Customers](https://knowledge.intershop.com/oh/go.php/iom/master/iom_en/topics/managing_customers/topic_searching_customers.html#task_buyer_search). -
On the top of the result list, click the Download
icon and select CSV download.
This exports the result set into a CSV file immediately and prompts you to download it.
