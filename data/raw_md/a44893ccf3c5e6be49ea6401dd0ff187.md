# Creating and Assigning Promotion Codes

This section explains how to enter promotion codes by hand and how to bulk-generate promotion codes. For information on how to import promotion codes from a CSV file, see [Promotion Data Import and Export](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/online_marketing/topic_promotion_data_import_and_export.html).

Creating promotion codes is performed on the Codes tab in the promotion code group details view.

## Entering Promotion Codes Manually

-
Select the channel from the context selection box, then select Marketing | Promotion Codes and switch to the Promotion Code Groups tab.
This displays a list of existing promotion code groups.
-
Click the promotion code group for which you want to create new promotion codes.
The promotion code group details view is displayed.
-
Select the Codes tab.
The Codes tab shows a list of promotion codes currently associated with the promotion code group.
-
Click the Add manually button.
A text box for entering promotion codes is displayed.
-
Enter the new promotion codes, then click Add.
Enter new promotion codes by hand or copy and paste them from another source into the text box. Each promotion code must be put on a separate line, must not exceed a maximum length of 128 characters, and has to be unique. When you click Add, the system reads in the promotion codes and automatically assigns them to the current promotion code group. If there are promotion codes that do not comply with the requirements mentioned above, the system will reject them and prompt you for action.

## Bulk-Generating Promotion Codes

-
Select the channel from the context selection box, then select Marketing | Promotion Codes and switch to the Promotion Code Groups tab.
This displays a list of existing promotion code groups.
-
Click the promotion code group for which you want to create new promotion codes.
The promotion code group details view is displayed.
-
Select the Codes tab.
The Codes tab shows a list of promotion codes currently associated with the promotion code group.
-
Click Generate, and provide the required information.
See the following table for details:
Table 1. Bulk-Generating Promotion Codes Option Description Promotion Code Prefix Each promotion code can be optionally prefixed with a user-defined fixed string. For example, if you enter "X-MAS_" the resulting promotion code will be "X-MAS_FD42P". Promotion Code Length Specifies the length of the variable part of the promotion code. The total length of the promotion code (the prefix plus the variable part) must not exceed 128 characters. No. of Promotion Codes Defines how many promotion codes the system will generate. Generating a large number of promotion codes may take a long time, depending on the available hardware and the current server load. Excluded Characters Specifies characters that are not to be used for promotion codes, e.g., potentially misleading, ambiguous characters like "l" or "I". Separate multiple characters with a comma. -
Click Generate.
The system generates the new promotion codes based on your specifications and assigns them to the current promotion code group.
