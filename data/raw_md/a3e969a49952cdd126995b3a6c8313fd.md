# Warranties and Gift Options

In Intershop Commerce Management, warranties, gift wraps and gift messages constitute special products with specific properties like, for example, not individually shippable or no inventory. Warranties, gift wraps and gift messages are grouped in dedicated system classification categories, which propagate the corresponding properties to these products when establishing the assignment relation.

[Catalog Types](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/concept_catalog_types.html).

As warranties are intended to be sold in combination with "real" products only, they feature specific pricing options, as outlined below.

| Option | Description |
|---|---|
| Fixed Price | Used to define a fixed amount to be charged as the warranty price. |
| Percentage | The warranty price depends on the product price. It is calculated as a percentage amount based on the list price of the product to which the warranty is applied. |
| Scaled Prices | The warranty price depends on the product price. This price type uses "brackets" to define value ranges to which specific warranty prices apply.
