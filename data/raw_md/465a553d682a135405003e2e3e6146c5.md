# Editing Page Templates

General page template editing options include modifying page template content, setting page template properties and copying (user-generated) page templates. Prior to being edited, page templates must be locked, see [Locking Page Templates](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_locking_page_templates.html#task_page_tmpl_lock).
