# Managing SEO Settings

Channel-specific SEO settings include the activation of Rich Snippets and locale-specific links for storefront pages.

[Intershop Organization Management User Manual | Managing Site Settings | Secure URL only server configuration](https://knowledge.intershop.com/oh/index.php/Display/3366K3).

## Enabling Rich Snippets for Product Pages

Intershop Commerce Management supports Rich Snippets for product pages, see [Rich Snippets](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/online_marketing/concept_rich_snippets.html).

To enable/disable Rich Snippets for product pages of this channel's storefronts:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select SEO Settings.
The SEO Settings preference page is displayed.
-
To enable Rich Snippets, select the Rich Snippets checkbox.
To disable Rich Snipptes, deselect the checkbox, respectively.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.

## Enabling Locale-Specific Links

[Enabling Locale-Specific Links](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_seo_settings.html#task_pref_gen_seo_locale).

To enable/disable locale-specific links:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select SEO Settings.
The SEO Settings preference page is displayed.
-
To enable language/region-specific links, select the Locale-specific Links checkbox.
To disable locale-specific links, deselect the checkbox, respectively.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.

## Enabling URL Rewriting

[URL Rewriting](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/online_marketing/concept_url_rewriting.html).

To enable/disable URL rewriting:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select SEO Settings.
The SEO Settings preference page is displayed.
-
To enable URL rewriting, select the URL Rewriting checkbox.
To disable URL rewriting, deselect the checkbox, respectively.
-
Specify a URL Rewrite Site Name.
Intershop recommends to use a readable name without specific characters.Note:Not specifying this value may cause some URL rewriting rules to malfunction. Contact your system administrator for details.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
