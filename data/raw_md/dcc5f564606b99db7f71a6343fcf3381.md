# Editing Objects in Batch Mode

The general approach for performing batch processes is similar for all object types. Be aware, however, that the set of tasks that can be performed in batch mode differs depending on the object type.

To edit objects in batch mode:

-
Locate the objects you want to edit.
- For details about locating products, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). - For details about locating pages, see
[Searching Pages](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_searching_pages.html#task_page_mgmt_search). - For details about locating includes, see
[Searching Includes](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_searching_includes.html#task_page_incl_search). - For details about locating page variants, see
[Searching Page Variants](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_searching_page_variants.html#task_page_vrnt_search). - For details about locating components, see
[Searching Components](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_searching_components.html#task_page_cmp_search).

-
Select the checkboxes of the objects to be edited and click the Edit Selected button.
Alternatively, perform a search and click the Edit All button to edit the search result.
-
Select the task and click Next.
This starts different sequences depending on the task to be performed.
-
Configure the task details as required and click Next.
Depending on the task to be performed, multiple steps may be necessary.
-
Confirm the operation and click Finish.
This completes the batch process for the selected objects.
