# Editing Sales Organization

-
In the navigation bar, select Sales Organizations.
A list of all sales organizations in the system is displayed.
-
Click the name of the sales organization you intend to edit.
The sales organization's details are displayed.
-
Under each tab edit the information as desired.
You can add, edit and delete information under each tab. The table below lists the available tabs and the information to be specified.
Table 1. Sales organization information Tab Information General Allows for specifying name, online status, ID and description for the organization. Contact Info Allows for specifying the default address data of the organization. Attributes Allows for defining custom attributes for the organization. Branding Allows for managing branding packages for the organization. Payment Allows for managing payment methods for the organization. Services Allows for managing additional services (if available) for the organization.
