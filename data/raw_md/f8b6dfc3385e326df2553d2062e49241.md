# Assigning All Products

To assign all products that are available in the channel repository to the product data feed configuration:

-
Select the channel from the context selection box, then select Catalogs | Product Data Feeds from the navigation bar.
A list of all configurations currently defined is displayed.
- Click the name of the configuration to which to assign products.
-
Change to the Products tab.
A list of all products currently assigned to the configuration is displayed.
-
Click Assign All.
All products that are available in the channel repository are included in the product data feed.

Note:

Newly created products will be included automatically in the product data feed when using Assign All.
