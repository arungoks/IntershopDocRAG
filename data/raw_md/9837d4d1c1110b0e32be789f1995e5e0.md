# Installing a Branding Package

To install a branding package for a partner:
-
Open the partner manager.
Select the partner channel from the context selection box, then select Partners. This displays the partner list.

-
Click the name or ID of the partner.
This displays the partner detail view.

-
Open the Branding tab.
This opens a list of all branding packages that have been uploaded.

-
Click Install in the row of the branding package you intend to install.
A list of all existing installations of this branding package is displayed.

-
In the New Installation field, select the locale for the installation and click Install.
