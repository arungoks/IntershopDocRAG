In [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management), the term query refers to a specific type of XML files that allow to dynamically construct SQL queries and to execute them in the [database](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214729187/database). The database response can be mapped to various Java objects, like primitives or [persistent object](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2167931462/persistent+object)s.
