Webforms are specific [development artifact](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214730413/development+artifact)s used to define forms for web pages, which may contain multiple input fields and and are subject to validation. Webforms are used together with [pipeline](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160821914/pipeline)s and [ISML template](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160822230/ISML+template)s - a pipeline is responsible to create and validate a form instance, and a template renders the web page.

In [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management), the webform [framework](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163970267/framework) allows to define forms for web pages, which may contain multiple input fields and and are subject to validation. [Webform](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214745494/webform) definitions are individual [development artifact](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214730413/development+artifact)s. Webforms are used together with [pipeline](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160821914/pipeline)s and [ISML template](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160822230/ISML+template)s - a pipeline is responsible to create and validate a form instance, and a template renders the web page.
