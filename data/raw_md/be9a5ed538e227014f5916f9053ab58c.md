This document replaces the outdated article with the ID 226N40 and the title *The Intershop Commerce Insight (ICI) Dashboard.*

The Intershop Commerce Insight (ICI) includes a Dashboard and the possibility to export and import Dashboards.

Dashboards can be configured in order to fit your needs.

As dashboards are unique for every user, there is the possibility to export and import them for sharing purposes.

The following steps describe how to import a dashboard

Attached you find an [example dashboard](https://knowledge.intershop.com/kb/index.php/Deliver/DOC/2882E6/CICDashExampleBoard.xml) file which can be used for import. It contains three basic reports: *Statistics per site*, *Top-level pipeline calls trend*, and *End user response time*. Feel free to import and modify as you wish.
