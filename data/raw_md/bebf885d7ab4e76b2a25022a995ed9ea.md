# Exporting Confirmed E-Mail Addresses

To export e-mail addresses that have been confirmed, e.g., in order to hand them over to an external newsletter tool:
-
Select the channel from the context selection box, then select Marketing | Newsletter.
This displays the Subscriptions tab with a table containing all registered e-mail addresses.

-
Click the Export Confirmed or the Export All button, depending on your E-Mail Confirmed setting in the Settings tab.
-
In the E-mail Newsletter Export dialog, specify a name for the export file and click Export.
A CSV file is created. This file can be downloaded on channel level under Marketing | Import & Export.
