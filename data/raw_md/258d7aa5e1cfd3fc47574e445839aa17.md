Generally, test [framework](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163970267/framework)s provide software testing methods that allow for automatically testing sets of one or more computer program modules together with associated control data, usage procedures, and operating procedures, to determine whether they are fit for use. [Intershop 7](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2162594838/Intershop+7) and [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management) support the following test frameworks:

Test cases are derived from [user stories](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163976840/user+story), [system use case](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152516473/system+use+case)s and [IAD](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163976934/IAD)s. They exist in the context of [system component](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2213813242/system+component)s and [release](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2159968461/release)s and are executed automatically or manually. A large number of successfully executed test cases generally indicates a good component and/or release quality. QA engineers own the testcases.

Automatic test cases rely on [test framework](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214730050/test+framework)s.

Test execution describes the action of running [test case](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163975282/test+case)s. The results are documented within the test management tool "Silk Central Test Manager" (SCTM). Failed tests normally lead to a new raised [defect](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2171864885/defect) (in the regarding Jira project).

Test execution planning is done in SCTM. Using this tool Quality Assurance defines all [progression test](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2164788411/progression+test)s and [regression test](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2164788409/regression+test)s (including [NFR](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2213818660/NFR) testing).
