Software as a service (SaaS) is a facet of [cloud computing](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2230485152/cloud+computing). It is a software licensing and delivery model in which software is licensed on a [subscription](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2227919888/subscription) basis and is centrally hosted.

The capability provided to the [customer-x] is to use the provider’s [application](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152517685/application)s running on a cloud infrastructure. The applications are accessible from various client devices through either a thin client interface, such as a web browser or an application [user interface](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2162594514/user+interface). The customer does not manage or control the underlying cloud infrastructure (network, servers, operating systems, storage, or even individual application capabilities), with the possible exception of limited user-specific application configuration settings.

Scrum is an iterative, incremental framework for project management often seen in agile software development, a type of software engineering.

A smoke build always produces a "full build", which means that the resulting build can be installed without having a previous build installed.
