# Classification Attributes

For categories of custom classification catalogs, classification attributes can be defined in order to predefine the attributes of the products assigned to a category. When assigning a product to a classification category with classification attributes defined, the attributes are offered to the product manager in a list. Note, however, that the product manager needs to set attribute values before the attributes are actually copied onto the product. If copied onto a product, the attributes become available via the Attributes tab which is part of the product detail view.
