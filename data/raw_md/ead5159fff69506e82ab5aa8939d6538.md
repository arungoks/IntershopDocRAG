# Setting a Default Category for Products

To define a default category for a product:

-
Locate the product for which you want to manage category assignments.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product.
This displays the Standard Product dialog. The Categories section of the General tab lists all categories to which the current product is assigned.
-
In the Categories section, select the category assignment you intend to set as default.
Select the radio button (in the Default Category column) in the row of the corresponding category assignment.
-
Click Apply.
The selected category assignment is immediately set as the default.
