# Accessing the Site Management Module

To access the site management module:

-
Select Site Management from the Intershop System Management navigation bar.
This opens the Sites overview page, which lists all sites served by the system and shows their Display Name, ID and Status.
-
Click a site name in the list to edit the settings for this site.
The site's detail page opens. It contains three tabs, General, Page Cache and Applications, which allow for setting the corresponding preferences, see
[Site Management Preferences](https://knowledge.intershop.com/oh/go.php/icm/master/smc_en/topics/site_management/topic_site_management_preferences.html#site_management_preferences).
