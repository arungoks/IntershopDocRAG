# Managing the Product Lifecycle

To set the last order date or end-of-life-date for a product:
-
Locate the product for which you want to set last order date or end-of-life-date.
-
Click the name of the product.
The General tab of the Standard Product dialog is displayed.

-
Select the Online/Offline tab.
-
Specify last order date and end-of-life-date as required.
If both dates are defined, make sure the end-of-life-date is after the last order date.
