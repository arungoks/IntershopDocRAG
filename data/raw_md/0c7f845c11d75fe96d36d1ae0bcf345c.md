With [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management), the relational database system is used to store data whenever a transactional access or search capability is required. As a rule of thumb, all business data and some of the system configuration data is stored in the relational database system. Access to the database system happens primarily via the object-relational mapping mechanism of the [ORM](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2167931836/ORM) engine, which internally uses the native client drivers to access the database server.

Currently, Intershop Commerce Management support Oracle database systems only.
