# Searching Intershop Commerce Management Users

Simple search and advanced search options are available. To search for specific users:
-
In the navigation bar, select Organization | Users.
This displays a list of all users in your sales organization.

-
Enter the user's name or ID in the search field.
To perform a more detailed search based on additional user information, switch to the Advanced Search. For example, you can search for a specific access privilege or a department.

-
Click Find.
This displays the search result in a list view.
