The IOM SOAP API to place an order silently ignores unknown shipping methods or maps them to default "Unknown Shipping Method" when configured accordingly.

As IOM Order REST API 2.0 will refuse orders with unknown methods unless that mapping is defined, you have to configure it before when using REST.

See [Guide - IOM Shop Onboarding | Carrier](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1917329114/Guide+-+IOM+Shop+Onboarding#Guide-IOMShopOnboarding-carrier) for more details.
