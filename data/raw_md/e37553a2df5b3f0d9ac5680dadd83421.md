# Notification Agent

The notification agent can be used by registered customers to receive an e-mail in case:

-
The price of a product has dropped to a certain value, or

-
A product which was out of stock is available again.


In Intershop Commerce Management, you can specify in which intervals the agent should check whether notification mails need to be sent out.
