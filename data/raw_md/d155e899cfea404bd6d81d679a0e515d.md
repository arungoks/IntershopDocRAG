# Managing CAPTCHA Usage

Note:

The general availability of a CAPTCHA service is controlled via the services configuration, see

[Services Management](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_services_management.html).-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select CAPTCHAs.
The details page is displayed.
-
Select the checkbox(es) for the intended storefront features.
In the reference storefront implementation, CAPTCHAs are available for the following features:
-
Redemption of gift cards & gift certificates

-
Registration

-
E-mail a friend

-
Forgot password

-
E-mail shopping cart

-
Contact us


-
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
