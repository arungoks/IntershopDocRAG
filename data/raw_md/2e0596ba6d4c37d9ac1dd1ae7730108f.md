# Content Upload Management

You can upload or delete content files either at the organization, channel, or application level. The Content Upload manager displays a content directory tree structure and a table showing the files in each folder. Content files can be specific to each locale. Before creating directories and uploading files, you must select the target locale.

*Content Manager*access privilege.
