# Process Chains

A Process Chain allows to execute a specific list of specific jobs in a predefined order. If you have to execute the same jobs in the same order more than once, it is the perfect tool to automate this. You can define the job list, the order of the job execution and a time-based trigger that allows to automate the execution of a process chain. You can execute the jobs serially or parallel or/and you can group different jobs.

Prerequisites for Process Chain Management:

-
The user needs the access privileges

*Process Chain Viewer*and*Process Chain Manager*.For more detailed information on how to achieve this, see

[Access Privileges and Roles](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_access_privileges_and_roles.html#con_org_acc). -
Log into the organization's Intershop Commerce Management application.

For example inSPIRED, the organization of Intershops blueprint shop.
