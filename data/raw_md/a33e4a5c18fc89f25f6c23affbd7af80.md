Common name for a front-end [application](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152517685/application) of the [sales channel](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2153813335/sales+channel), designed to sell directly to [customer](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2153810357/customer)s. The storefront represents a [touchpoint](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163980932/touchpoint).

Generally, a web shop is a type of [online shop](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2179410694/online+shop) whose [storefront](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2178600227/storefront) [application](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152517685/application) is designed to be used with web browsers from usual desktop/laptop computers.
