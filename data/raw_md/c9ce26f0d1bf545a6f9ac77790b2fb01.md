# Running Product Data Feeds Automatically

Besides manual execution, product data feed processes can also be triggered via a scheduled job.
-
Select the channel from the context selection box, then select Catalogs | Product Data Feeds from the navigation bar.
A list of all configurations currently defined is displayed.

-
Click the name of the configuration.
-
Change to the Scheduling tab.
-
Specify the execution interval for the current configuration, then click Apply.
