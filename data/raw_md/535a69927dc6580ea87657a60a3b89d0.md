The function `stringToXml()`

returns the result of a string expression after converting any special characters, such as such as `<`

, `>`

, `&`

and `&`

into their appropriate XML encoding sequences. For more information see [Encoding Expression Results](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775688409/Reference+-+ISML+Expressions#Reference-ISMLExpressions-EncodeExpressResult).

stringToXml(<operation>);

This example shows that although the automatic encoding of [<ISPRINT>](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775992843/ISML+Tag+-+ISPRINT)has been disabled by setting encoding to off, you can encode the result of an expression by explicit use of the function `stringToXml()`

.

<ISPRINT value = "#stringToXml('<<<')#" encoding = "off"> // returned string: "<<<"
