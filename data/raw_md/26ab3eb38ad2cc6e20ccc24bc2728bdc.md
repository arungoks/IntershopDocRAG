In software, a dashboard is generally a real-time [user interface](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2162594514/user+interface) that combines easy-to-read information about a [user](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160822248/user)'s tasks, business KPIs, etc.

In [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management), a dashboard is a customizable set of views ([widget](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2190214639/widget)s) that display certain data and allow to navigate to the corresponding [back office](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2159969605/back+office) area.

*A widget in general is a (small) frame on a page (e.g. place on a dashboard) that contains information about one specific topic, with the purpose to give a quick overview about relevant data. In Intershop Commerce Management, it functions as the presentation of most relevant information derived from Intershop data in an easy to read way and possibly provides access to system functionality relevant in the given data or task context.*
