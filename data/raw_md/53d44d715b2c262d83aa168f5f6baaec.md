# Managing Product and Category Images

This section describes the tasks involved in managing product and category images. The image management is divided into four tabs: Image Upload, Image Views, Image Types and Image Sets, which are explained below.

| Tab | Description |
|---|---|
| Image Upload | Used to create the image directory structure and to upload the image files to the Intershop Commerce Management system. |
| Image Views | Used to manage the image views available in the organization. |
| Image Types | Used to manage the image types available in the organization. |
| Image Sets | Used to manage the image sets available in the organization. |

For details about image meta data import/export, see [Exporting and Importing Image Meta Data](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/topic_exporting_and_importing_image_meta_data.html#task_pim_impex_imageMeta).

Note:

The tasks described in this section are performed in the context of the organization or a partner channel.
