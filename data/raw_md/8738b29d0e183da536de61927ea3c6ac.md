# Breaking Changes

- Webpack 5 upgrade - custom webpack configuration has to be updated if customized.
- Build process in docker image was enhanced to enable multiple theme configurations at runtime.
- The
* environment.ts* handling and with that the handling of *environment.local.ts* was changed. See [Migrations / 0.30 to 1.0](https://github.com/intershop/intershop-pwa/blob/develop/docs/guides/migrations.md#031-to-10) for more details. - Locale definitions in
*environment.ts* models are no longer supported, only ICM channel configurations are now used for switching locales.
