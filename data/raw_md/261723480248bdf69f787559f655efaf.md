# Navigation

After logging in, the main screen of Intershop Organization Management is displayed.

This screen is sub-divided into two areas: a navigation bar on the left and the main page area on the right, which displays lists or dialogs associated with the module selected in the navigation bar.

The navigation bar comprises the following sections:

-
[Sales Organizations](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_organizations/topic_managing_organizations.html)Allows for creating and managing sales organizations.

-
[Users](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_administration_users/topic_managing_administration_users.html)Allows for creating and managing administration users.

-
[Classification Catalogs](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_classification_catalogs/topic_managing_classification_catalogs.html)Allows for creating and managing classification systems.

-
[Import & Export](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_data_import_and_export/topic_managing_data_import_and_export.html)Allows for importing and exporting classification systems.

-
[Site Management](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_site_settings/topic_managing_site_settings.html)Allows for controlling site-specific settings.

-
[Data Replication](https://knowledge.intershop.com/oh/go.php/icm/master/oma_en/topics/managing_data_replication_processes/topic_managing_data_replication_processes.html)Allows for controlling and monitoring data replication processes.

-
Allows for creating and editing transport configurations.


In addition, the quick access buttons located above the main page area provide immediate access to certain functionality:

-
Home

Opens the front end start page.

-
Profile

Opens the user detail view for the current user (

*admin*) to edit general and address information. -
Help

Opens the Intershop Organization Management User Manual.

-
Logout

Logs out the user (

*admin*) from the current session and displays the log on screen.
