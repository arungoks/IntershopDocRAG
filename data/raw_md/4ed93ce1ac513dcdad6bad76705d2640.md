# Modifying Variation Type

To modify the details of an existing product variation type:
-
Select the intended management context from the context selection box, then select Catalogs | Variation Types.
This displays a list of all variation types currently defined for the current repository.

-
Click the name of the variation type you want to edit.
This opens the variation type detail view.

-
Modify the details as required.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
