# Getting Started

The Intershop Admin Center provides basic features designed to help administrators handling the user management and the assignment of access privileges.

Being the main actor in the Admin Center, the administrator invites other users and provides them with exactly the apps and modules they need for their daily business.

[Getting Started](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/getting_started/topic_iac_getting_started.html)This section provides general information about the Admin Center. Moreover, it covers the topics of navigation, and apps and modules.

[User Management](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/topic_user_management.html)This section provides instructions for managing users in the Admin Center.

[Apps & Modules](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/apps_modules/topic_apps_modules.html)This section provides guidance on managing apps and modules in the Admin Center.

[Connection Management](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/connection_management/topic_connection_management.html)This section provides instructions for managing connections in the Admin Center.

[Glossary](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/references/ref_iac_glossary.html)This section provides a glossary explaining terms used throughout the application and its modules.
