# Search Index Services

To provide powerful storefront search options, like full-text search or dynamic navigation, Intershop Commerce Management integrates third-party search engines via dedicated adapter cartridges. Once implemented and enabled, the integrated search builds and queries specific index files located in the site's file system structure.

For details about the search services management, see [Managing Search Index Services](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_search_index_services.html#task_pref_servSearch).

With an integrated search engine, the Mass Data section of Intershop Commerce Management integrates a dedicated Search Index manager. Use this manager to enable, configure and build the search indexes, which then serve as the base for the intended advanced search options in the storefront.

For details about search index management, see [Managing Search Indexes](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_search_indexes.html).
