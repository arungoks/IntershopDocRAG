A new job to clean up search indexes is available from 7.10.27.0, see [Job - Cleanup Search Indexes](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1922251158/Job+-+Cleanup+Search+Indexes). The cleanup of Solr indexes requires a Solr adapter version of 2.3.0, see [Public Release Note - SolrCloud Search Service Adapter 2](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1891537964/Public+Release+Note+-+SolrCloud+Search+Service+Adapter+2).

Furthermore, two new process chains are available: [Job - ProcessChain.InitIndexesProductAssignments](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1922251221/Job+-+ProcessChain.InitIndexesProductAssignments) and [Job - ProcessChain.RebuildIndexesProductAssignments](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1922251197/Job+-+ProcessChain.RebuildIndexesProductAssignments).

Run *dbmigrate* for adding the job configurations to the SLDSystem schedules.
