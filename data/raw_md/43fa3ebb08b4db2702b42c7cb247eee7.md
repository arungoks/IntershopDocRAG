# Google Tag Manager

Google Tag Manager is a tool to manage web page tags, including code snippets for conversion tracking, site analytics and others. Usually, the tags maintained with Google Tag Manager are to be included manually in the intended web pages. Intershop Commerce Management integrates Google Tag Manager using a dedicated service, which does this job.

For general information about Google Tag Manager, refer to [Google Tag Manager website](https://www.google.com/analytics/tag-manager/).

For details about setting up Google Tag Manager with Intershop Commerce Management, see [Using Google Tag Manager Service](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_using_google_tag_manager_service.html#task_pref_servGtm).
