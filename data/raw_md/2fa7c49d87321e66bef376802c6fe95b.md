In ICM 7.10 there is a [known issue](https://knowledge.intershop.com/kb/go.php/o/89883) with Google Chrome 116/117. If a web page uses mixed protocols (http and https), Google Chrome blocks javascript with https (e.g. add to cart) on http pages (e.g. product), because [http://domain](http://domain) is not the same site as [https://domain](https://domain.).

We recommend to use only https for all web pages. For configuration details, see the [Guide - Secure URLs Only](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/48581247408/Guide+-+Secure+URLs+Only).
