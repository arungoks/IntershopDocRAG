# Promotion Templates

Marketing managers can create promotions manually or based on a (predefined) set of promotion templates. Using templates saves time since it is a one-page setup for simple promotions.

Currently, Intershop Commerce Management provides the following promotion templates:

| Template | Description | Example |
|---|---|---|
| Free Shipping on Order Value | Customers get free shipping if the order value reaches the minimum order value defined by the marketing manager. | Get free shipping for orders $100+. |
| Fix Discount on Order Value | Customers get a fix discount if the order value reaches the minimum order value defined by the marketing manager. | Get $15 off for orders $200+. |
| Percentage Discount on Order Value | Customers get a percentage discount if the order value reaches the minimum order value defined by the marketing manager. | Get 10% off for orders $100+. |
