# Logging

This section describes how to change the logging options of the Intershop Commerce Management application.

Using the Intershop System Management Logging module, you can customize the existing logging appenders defined for your Intershop Commerce Management instance. The settings specified in this Intershop System Management module are written to <IS.INSTANCE.SHARE>/system/config/cluster/logging.properties.
