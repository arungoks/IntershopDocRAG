IOM Helm Charts 2.0.0 or newer are required to operate IOM 4.0.

For more information, refer to [Public Release Note - IOM Helm Charts 2.0](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/47844426739/Public+Release+Note+-+IOM+Helm+Charts+2.0)

The following argument was added to Java Parameters: `-Dfile.encoding=UTF-8`

If your system sets this parameter otherwise, this setting should be removed.
