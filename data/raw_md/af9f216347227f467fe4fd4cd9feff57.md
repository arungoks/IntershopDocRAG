# Assigning Links to Link Group

To assign links to a link group:
-
Open the Link Groups manager.
-
Select the channel from the context selection box, then select Marketing | Link Management.
-
Click Link Groups.

This displays a list of existing link groups.

-
Click the link group to which links are to be assigned and change to the Links tab.
This displays the list of links that are already assigned to this link group.

-
Click New.
A list of short links defined for this management context is displayed.

-
Select the links to be assigned to the link group and click Assign.
This assigns the selected short links and returns you to the Links tab of the link group detail view.
