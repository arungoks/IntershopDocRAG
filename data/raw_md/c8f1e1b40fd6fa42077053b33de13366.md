The wireframes listed here contain interaction designs for storefront pages. In particular, this involves the functionalities of the Cybersource Service Connector and its implementation in the corresponding Blueprint Store of the Intershop e-commerce platform. Full documentation of the Intershop reference application can be found in the wireframes of the corresponding version of the Intershop Progressive Web App.

Each ZIP file contains a clickable prototype. Open *HTML/index.html* to see the start page. From there you can navigate directly to other pages via the sitemap.

There are features on some pages which are interactive. Use the 2nd icon next to the Axure logo and select the option "Show Hotspots".

The design is responsive in the same way the starter store is.

To see the different layouts, you can resize your browser, or you can use the drop-down to switch between the views.

The prototype is described with page and widget notes. Use the 1st icon to show the documentation.
