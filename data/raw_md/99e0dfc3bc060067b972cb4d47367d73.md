# Uploading Branding Package

To upload a new branding package for a partner:
-
Open the partner manager.
Select the partner channel from the context selection box, then select Partners. This displays the partner list.

-
Click the name or ID of the partner.
This displays the partner detail view.

-
Open the Branding tab.
This opens a list of all branding packages that have been uploaded.

-
Click Upload.
The New Branding dialog is displayed.

-
Provide a name for the branding package, a description, and locate the branding archive file.
-
Click Apply.
The new branding package is uploaded and can now be installed.
