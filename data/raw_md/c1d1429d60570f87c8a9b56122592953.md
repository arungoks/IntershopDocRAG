# Breaking Changes

- Removed the
`not_available.png`

image, and replaced it with an SVG image (without text inside the image), and adjusted file references (see [Migrations / 3.0 to 3.1](https://github.com/intershop/intershop-pwa/blob/develop/docs/guides/migrations.md#30-to-31) for more details). - Changed the default queryParamsHandling for product detail links via product image or product name component (see
[Migrations / 3.0 to 3.1](https://github.com/intershop/intershop-pwa/blob/develop/docs/guides/migrations.md#30-to-31) for more details). - Removed the
`equalToControl`

validator. Use the `equalTo`

validator instead. Find more information in the method description in `special-validators.ts`

.
