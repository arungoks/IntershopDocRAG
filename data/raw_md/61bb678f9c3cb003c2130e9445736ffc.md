A new job to recalculate the sales target actual amount for all contracts was introduced, see [Job - Recalculate the Sales Target Actual Amount for all Contracts](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1911925467/Job+-+Recalculate+the+Sales+Target+Actual+Amount+for+all+Contracts).

This is useful in case the order status was changed to CANCELLED. It should be run after the order status import.

Run *dbmigrate* for adding the job configuration to the schedules.
