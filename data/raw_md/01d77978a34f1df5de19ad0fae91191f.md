# Managing Regional Settings at Channel Level

Channel administrators can define default locale-specific settings for the channel storefront applications.

Note:

Settings defined here can be overridden at application level, see

[Managing Regional Settings at Application Level](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/task_managing_regional_settings.html)).To manage the default regional settings for the channel storefronts:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select Regional Settings.
The details page is displayed.
-
Edit the settings as required.
The table below lists the available parameters.
Table 1. Channel regional settings Parameter Description Default Unit of Mass Defines the default unit of mass for the channel storefronts. Default Unit of Length Defines the default unit of length for the channel storefronts. Default Timezone Defines the default timezone for the channel storefronts. -
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
