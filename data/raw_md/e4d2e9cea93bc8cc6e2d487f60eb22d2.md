# Viewing User Details

-
Click User Management in the main content area on the welcome screen, or in the side navigation.
The
*User Management*overview page opens. Here you can see a list of all users. -
Click on a user in the user list.
This displays the detail page of the selected user with a list of the assigned apps, modules and privileges.

- Username
The name of the user.

- Apps and modules
Here you can see all apps and modules that are assigned to the user. Furthermore, you can add apps or modules, see

[Adding Apps and Modules](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/topic_adding_apps_and_modules.html#topic_adding_apps_and_modules), and remove apps or modules, see[Removing Apps or Modules from a User](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_removing_apps_or_modules_from_a_user.html). - Access privileges
Here you can see which role has been assigned to the user and a description.
