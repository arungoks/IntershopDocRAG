# Running a Recommendation Data Feed Automatically

Besides manual execution, recommendation data feed processes can also be triggered using a scheduled job.
-
Select the channel from the context selection box, then select Marketing | Recommendations - Data Feeds.
This displays a list of all feed configurations that are currently available.

-
Click the name of the configuration.
-
Change to the Scheduling tab.
-
Specify the execution interval for the current configuration, then click Apply.
Otherwise, click Reset to discard your changes.
