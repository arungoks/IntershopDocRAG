# Order Tracking

Orders can assume various order states which enable order managers to track which stages of the order handling process an order has already passed. Order states are modified upon importing order states, see [Order Import and Export: Concept](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/concept_order_export_and_import.html).

Various filters are available to track down orders depending on their state.

## Order States

The order state reflects the state of the entire order, namely:

-
New

-
In progress

-
Canceled

-
Canceled and exported

-
Not deliverable

-
Delivered

-
Returned

-
Pending


## Fulfillment States

The fulfillment state reflects the state of the fulfillment of the products ordered by customers. It is displayed as part of the order details. Fulfillment states include:

-
New

-
In progress

-
Canceled

-
Canceled and exported

-
Partially delivered

-
Delivered


## Payment States

Customers can place orders using various payment methods, like credit cards, and combining them, for example, with gift certificates. As part of the order details, order managers can display the state of each payment transaction used for an order. Payment states include:

-
Unprocessed

-
Partially captured

-
Captured

-
Voided authorization failed

-
Settled

-
Refunded

-
Refund failed
