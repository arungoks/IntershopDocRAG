# Importing Short Links

-
Navigate to the Link Import & Export page of Link Management.
- Select the channel from the context selection box, then select Marketing | Link Management.
- Click Import & Export.

-
Upload the file to be imported.
For details, see
[Uploading Short Links for Import](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/online_marketing/task_uploading_import_files3.html). -
On the Import & Export page, click one of the Import buttons according to the uploaded file format.
A list of available import files is displayed.
-
Select the file to import, then click Next.
If you intend to import CSV formatted data, specify the following additional import parameters:
-
Mapping template

-
Field delimiter

-
Format line number

-
Custom format line


-
-
Click Next, then select the import mode.
For importing links, the import modes UPDATE and REPLACE are available. For import mode details, see

[Standard Import Modes](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/concept_catalog_import_and_export.html#con_cat_impex_modes).Note:The import mode DELETE for short links is available from Intershop Commerce Management 12.0. -
Click Import.
The import process is started and you are returned to the Import & Export page where you can review the status of the import process.
