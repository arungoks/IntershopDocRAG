This overview includes documentation about the Business Tracking Framework in Intershop Commerce Management. It details how the framework collects and provides event data to third-party applications, such as web analytics and recommendation engines. The framework defines triggerable events, provides components for developers, and offers interfaces and managers for integrating with these engines.
