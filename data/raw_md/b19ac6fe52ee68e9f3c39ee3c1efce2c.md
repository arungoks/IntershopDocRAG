To display the available includes for a specific context:

Select the management context from the context selection box.

This opens the navigation bar of the management context.

In the navigation bar, select Content | Includes.

This opens the Include list view.

Select the application type (optional).

On channel or organization level, the list shows all includes of all application types available in the current context. Selecting the application type defines the content model you work with (see What Is an Application?) and filters the includes accordingly.

Tip:

This setting is active until you log off from Intershop Commerce Management. To undo the selection, select All or another application type.
