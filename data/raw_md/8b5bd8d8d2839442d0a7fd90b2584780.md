# Editing Classification Catalog

To edit a classification catalog:

-
In the navigation bar, select Classification Catalogs.
A list of all global classification catalogs in the system is displayed.
- Click Edit next to the catalog you intend to edit.
- Modify the details as required.
- Click Apply to save your settings.
