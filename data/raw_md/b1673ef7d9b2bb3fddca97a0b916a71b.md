# Creating Coupons for Sending to a Customer

This section provides information on creating coupons and sending them via e-mail.

[Redeeming Coupons](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/managing_orders/task_redeeming_coupons.html).

Coupons entitle customers to get the benefit of a specific promotion if all defined conditions are met.

While resolving a customer inquiry in a contact session, see [Starting a Contact Session](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/task_starting_a_contact_session.html), you can create a new customer-specific coupon for an existing promotion and send it to the customer.

For information about creating and managing promotions, refer to the [Intershop Commerce Management User Manual | Promotion Management](https://knowledge.intershop.com/oh/index.php/Display/32647Y).

To create a new coupon:

-
Start (if not already running) a contact session, see
[Starting a Contact Session](https://knowledge.intershop.com/oh/go.php/iap/master/cec_en/cec/contact_sessions/task_starting_a_contact_session.html). - Open the Coupons tab.
-
Select the promotion for which you want to create a coupon.
Note:The promotion select box displays only promotions set up accordingly in Intershop Commerce Management.
-
Click Create coupon.
The according coupon is generated immediately and displayed in the list.
-
Click the
*E-Mail*icon () in the Action column of the coupon section.This submits the generated coupon to the customer via e-mail.Tip:Once a coupon is sent out, you can still re-submit it using the*E-Mail*icon ().
