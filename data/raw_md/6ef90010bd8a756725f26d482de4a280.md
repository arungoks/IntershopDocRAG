# Viewing Order Promotions

To view the promotions that apply to an order:
-
Locate the order you want to view.
-
Click the ID of the intended order.
This opens the Order Detail view.

-
Change to the Promotions tab.
This displays all promotions that apply to the current order, grouped by types (order promotions, shipping promotions, line item promotions). For each applied promotion (group), the table shows the (total) discount amount. To view the details of a promotion, click the name of the promotion. This opens the promotion in the detail view.
