The <ISUUID> tag can be used to generate a single UUID string for a given name (object).

<isUUID ( name = "( {simple name} )" )>

<ISUUID name="CategoryRenderEntityID">

`name`

The key under which the UUID value is stored in the pipeline dictionary.
