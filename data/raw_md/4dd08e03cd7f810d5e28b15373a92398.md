# Glossary

| Term | Description |
|---|---|
| Component | Components are the container objects that hold the actual content to be displayed on the final web page, i.e., a text, an image, a specific UI control, etc. |
| Design View | The Design View provides a graphical interface for managing and editing your online shop’s content directly within the context of the live page. You can preview changes instantly as you make them. The Design View allows you to assign components to slots, replace images, update data, and perform other content editing tasks. |
| Intershop Content Management Center | The Content Management Center is part of the Intershop Admin Portal and is focused on managing the content of your online shop. |
| Publishing | This component setting allows you to determine the global visibilty of a component. If a component is published, it is visible across all contexts where it is used, i.e., on all pages where it is assigned to. If unpublished, it is not displayed anywhere in the shop during the specified time frame. |
| Visibility (Hidden) | This component setting determines whether a component is visible or hidden on the current page or placement only. If a component is hidden, this is only the case in the context you are applying the setting. On all other pages where this component is assigned as well, it remains visible. |
