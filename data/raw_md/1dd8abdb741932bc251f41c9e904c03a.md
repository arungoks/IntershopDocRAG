# Editing Component Templates

General component template editing options include modifying component template content, setting component template properties and copying (user-generated) component templates. Prior to being edited, component templates must be locked, see [Locking Component Templates](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_locking_component_templates.html#task_page_cmpTmpl_lock).
