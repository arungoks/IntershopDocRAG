# Hiding Default Component Content

[Component Template Management](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_component_template_management.html)) may propagate placeholders with default content to derived components. When editing components, you can hide this default content. To do so:

-
Select the management context from the context selection box, then select Content | Components.
This opens the Components list view.
-
Click the name of the component you intend to edit.
This opens the Content tab of the detail view.
-
In the placeholder section, click the icon next to the intended placeholder.
This opens the settings dialog box.
-
Select the Hide default content checkbox, and click Apply.
The content as derived from the template will not longer be displayed.Tip:In components that are shared from a parent management context (see
[Content Asset Sharing](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_content_asset_sharing.html)), the option to hide default content is not enabled.
