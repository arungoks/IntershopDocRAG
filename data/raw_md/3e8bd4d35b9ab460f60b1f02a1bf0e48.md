# Exporting Products

Note:

To export products:Currently, the Order Management Tool supports CSV file export only.

-
Search for the products. For details, see
[Searching Products](https://knowledge.intershop.com/oh/go.php/iom/master/iom_en/topics/managng_products/topic_searching_products.html#task_item_search). -
On the top of the result list, click the icon and select CSV download.
This immediately exports the result set into a CSV file and prompts you to download it.
