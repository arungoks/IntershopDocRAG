# Deleting Intershop Commerce Management Users

-
In the navigation bar, select Organization | Users.
This displays a list of all users in your sales organization. If the user you are looking for is not displayed, use the search functionality to find the user, see
[Searching Intershop Commerce Management Users](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/task_searching_intershop_commerce_management_users.html). - Select the checkbox of the user to be deleted.
-
Click Delete, and confirm the operation.
This removes the selected user from the system.

Tip:

You can also delete a user by clicking Delete in the User dialog.
