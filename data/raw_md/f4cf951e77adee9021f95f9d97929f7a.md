The following table lists all states of *Document Status Model* with IDs and short descriptions.

| State | ID | Description |
|---|---|---|
| INITIAL | 0 | Document was initially stored in the database |
| CHECKED | 1000 | Document was checked |
| TRANSMISSION_CREATED | 3100 | Transmission was created |
| PROCESSED | 5000 | Document was processed/ created |
| INVALID | 6000 | Validation of the document failed |
| CLOSED | 6200 | Document cannot be used any longer |
