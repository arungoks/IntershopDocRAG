| Name, ID, Description |
Enter a name, a unique ID string, and a descriptive text for the promotion code group. |
| Active (check box) |
Select the Active check box to enable the promotion code group. As long as the promotion code group is not enabled, the promotion codes assigned to the group will not be accepted by the system when entered in the storefront. |
| Activation |
Specify an activation period for the promotion code group. If you want the promotion code group to be enabled as long as the promotion runs it is assigned to, select the Active during promotion period check box. Otherwise define a differing activation period for the promotion code group. |
| Redemption |
Define how many times a promotion code can be redeemed, either in regards to a single customer or to all customers. Promotion Code reuse can be either unlimited or limited to a specific number. Note: In B2B scenarios, Promotion Code Reuse per Customer refers to actual B2B customers, not to individual users of a customer.
|
| Promotion Type |
When you select the In Store Redemption check box, customers can redeem a promotion code not only online, but also at their local store. |
