The command line client (CLC) is a tool to trigger pre-configured [job](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214729603/job)s or run import processes in the context of [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management). The CLC is self-contained, that is, it does not require any other parts of the Intershop instance. The CLC can be copied to and run from any other machine with a JRE installed.
