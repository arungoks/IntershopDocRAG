# Accessing Intershop System Management

Intershop System Management is a web front application running in a web browser.

To access Intershop System Management via a standard web browser:

-
Enter a URL with the following syntax:
http://<SYSTEM.HOST>/INTERSHOP/web/BOS/SMC
where <SYSTEM.HOST> is either the IP or DNS name of the machine hosting the Intershop Commerce Management Web Gateway.

-
Log in as administrator.
In a default Intershop Commerce Management installation, use the following credentials:

-
Login:

*admin* -
Password:

*!InterShop00!*Warning:Intershop strongly recommends to change the default administration user password at the first login. For details, see[Changing the Intershop System Management Password](https://knowledge.intershop.com/oh/go.php/icm/master/smc_en/topics/getting_started/task_changing_the_system_management_password.html).
