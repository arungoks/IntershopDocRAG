# Viewing Page Variants

-
Select the management context from the context selection box.
This opens the navigation bar of the management context.
-
In the navigation bar, select Content | Page Variants.
This opens the Page Variants list view.
-
Select the application type (optional).
On channel or organization level, the list shows all page variants of all application types available in the current context. Selecting the application type defines the content model you work with (see
[What Is an Application?](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/concept_what_is_an_application.html)) and filters the page variants accordingly.Tip:This setting is active until you log off from Intershop Commerce Management. To undo the selection, select All or another application type.

[Page Variant Manager](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/concept_page_variant_manager.html).
