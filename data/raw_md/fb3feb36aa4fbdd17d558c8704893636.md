# Assigning Products by Search Query

-
Select the channel from the context selection box, then select
Catalogs | Product Data Feeds
from the navigation bar.
A list of all configurations currently defined is displayed.
- Click the name of the configuration to which you want to assign products.
-
Change to the Products tab.
A list of all products currently assigned to the configuration is displayed.
-
Click Assign by Search.
The product list view is displayed.
-
Enter a search query in the search field and click Find.
The products matching the search query are listed.
-
Click Assign Search Query.
The products matching the search query are assigned.

Note:

Newly created products will be included automatically in the product data feed when using Assign Search Query.
