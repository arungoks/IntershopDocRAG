# Creating Link Groups

To create a link group for short links:
-
Open the Link Groups manager.
-
Select the channel from the context selection box, then select Marketing | Link Management.
-
Click Link Groups.

This displays a list of existing link groups.

-
Click New.
This opens the New Link Group dialog.

-
Provide a name and a description for the link group.
-
Click Apply.
This creates the new link group. You can now associate links with this group.
