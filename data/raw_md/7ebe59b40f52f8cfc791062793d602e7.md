# Editing Connections

This section describes how to adjust an existing connection. This applies to both ICM and PWA type connections. To adjust an existing connection:

-
Click Connection management in the main content area on the welcome screen, or in the sidebar.
The
*Connection management*overview page opens. Here you can see a list of all connections. -
Click the
*Edit*icon () of the connection in question.The*Edit <connection ID>*dialogue opens. -
Make your desired adjustments to the connection.
For information on the configuration parameters, see
[Custom Attributes](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/connection_management/concept_custom_attributes.html). -
Click the Save changes button (or discard this process by clicking the Cancel button).
The dialog closes and a confirmation message appears.
