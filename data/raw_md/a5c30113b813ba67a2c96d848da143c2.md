# Localization

Channel managers can maintain the static texts to be displayed in the sales channel storefront on a channel-specific base and separately for each available locale. The Localization Manager allows for searching and editing individual storefront texts. In addition, you can import localized values and export existing texts for localization.

For details about the localization management, see [Localization Management](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_localization_management.html).
