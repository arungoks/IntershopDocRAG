Intershop Copilot for Buyers is an AI-powered purchase and service assistant integrated within a Progressive Web Application (PWA), allowing users to access intelligent recommendations, search capabilities, and personalized actions by connecting various APIs, data sources, and AI models. Intershop Copilot for Buyers centralizes and streamlines user interactions by combining:

User-specific data retrieval through APIs

Intelligent search powered by AI

Actionable recommendations and insights generated in real time

This approach enhances productivity, simplifies workflows, and personalizes user experiences.

This concept is aimed at business users, product managers, and developers.

The general approach revolves around:

**Integration**: Seamlessly combining user data from external APIs (ICM and Product APIs) with AI-powered services.

**Indexing and Searching**: Using an indexing service to organize data for fast and intelligent retrieval.

**AI Recommendations**: Leveraging a Copilot AI system to interpret user input and provide context-aware suggestions or actions.

**User-Centric Design**: Delivering these features through an intuitive PWA interface to ensure accessibility and usability.

Intershop Copilot for Buyers is mainly based on FlowiseAI. For a detailed description, see the [official Flowise documentation](https://docs.flowiseai.com/).

Intershop Copilot for Buyers acts as the central AI assistant within the storefront, orchestrating product discovery and contextual assistance for buyers. It connects to the shop’s existing search infrastructure and can retrieve products using the configured search provider of the commerce platform (e.g., SOLR, SPARQUE, or other search engines).

This means that the Copilot works with the standard search capabilities of the Intershop Commerce Platform and is not tied to a specific search technology.

If advanced search and personalization services such as [SPARQUE.AI](http://SPARQUE.AI) are integrated, the Copilot can additionally leverage these capabilities, for example personalized rankings, behavioral relevance, or recommendation signals.

When a user is logged in, the Copilot can also access customer-specific information from the Intershop Commerce Management (ICM), such as order history or account context. In addition, shop content (e.g., guides, help pages, or use-case descriptions) can be incorporated through vector-based AI search to provide richer, contextual answers.

The Copilot Client is a Progressive Web App (PWA) extension that provides the frontend chat interface. Within the client, all actions that the Intershop Copilot for Buyers can perform in the PWA are defined, such as displaying a product detail page.
