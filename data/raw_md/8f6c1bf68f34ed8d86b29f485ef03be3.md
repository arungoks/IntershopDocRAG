# Viewing Module Details

You can access the details of a module from the user detail page. To view module details:

-
Navigate to the user detail page, see
[Viewing User Details](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_viewing_user_details.html). -
Click the arrow icon () next to an app name.
This will unfold the row, revealing the related modules.
-
Click on a module (for example,
*User management*).The detail page of the selected module opens.
