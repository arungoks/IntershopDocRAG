# Assigning Users to Modules

You can assign existing users to a module. Existing means that the user has already been added to the Intershop Administration Portal and can be found in the user list. For details, see [Adding Users](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_adding_users.html#task_inviting_users).

To assign users to a module:

-
In
*Apps & modules*, click on the module of an app, for example User management. For details, see[Viewing Apps and Modules](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/apps_modules/task_viewing_the_apps_and_modules_overview.html).The detail page of the selected module is displayed. -
Click + Add (or + Add user in case no users have been assigned to this module yet).
The corresponding dialog with a user search opens.
- Select one or more users from the list or enter the username or e-mail address into the search input field.
-
Click the Add button (or discard this process by clicking the Close button).
The dialog closes and a confirmation message appears. The number of available seats is decreased. The added user receives an notification e-mail.
