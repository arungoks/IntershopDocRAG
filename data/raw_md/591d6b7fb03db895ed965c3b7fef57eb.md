# Editing Pages in Batch Mode

Certain page management tasks can be performed on multiple pages at once via batch processes. Tasks that can be performed in batch mode include:

-
Assigning page variants to selected pages

-
Setting a parent to the selected pages

-
Assigning a label to the selected pages

-
Deleting the selected pages


For details about editing pages in batch mode, see [Editing Objects in Batch Mode](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/task_editing_objects_in_batch_mode.html).
