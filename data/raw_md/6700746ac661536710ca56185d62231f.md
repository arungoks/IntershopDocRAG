# Editing Intershop Commerce Management User Information

To edit an existing user:
-
In the navigation bar, select Organization | Users.
-
Click the name of the user you want to edit.
This opens the user detail view.

-
On each tab, edit the user information as required.
Available information includes attributes, access privileges, department roles and addresses. Save the changes you make on one tab before changing to the next tab.
