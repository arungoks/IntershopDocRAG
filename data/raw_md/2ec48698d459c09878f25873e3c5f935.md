# Uploading Promotions for Import

-
Select the channel from the context selection box, then select Marketing | Import & Export.
This displays the Promotion Import & Export manager.
-
In the Files section of the Import & Export manager, click Upload.
A list of files that have already been uploaded is displayed.
- In the Upload File field, specify the file path or use the Choose File button to locate the file.
-
Click Upload.
The uploaded file is displayed in the list. It can now be imported.

Tip:

In order to save system resources, Intershop recommends to delete the uploaded import files after the import process has been completed.
