# Editing Page Variants in Batch Mode

Certain page variant management tasks can be performed on multiple page variants at once via batch processes. Tasks that can be performed in batch mode include:

- Setting publishing period of the selected page variants
- Assigning the selected page variants to pages
- Assigning a component to the selected page variants
- Assigning label to the selected page variants
- Deleting the selected page variants

For details about editing page variants in batch mode, see [Editing Objects in Batch Mode](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/task_editing_objects_in_batch_mode.html).
