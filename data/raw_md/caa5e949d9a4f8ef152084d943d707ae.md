# Activating All Shared Products

To activate all products shared from a master repository for the channel repository:
-
Select the channel from the context selection box, then select Catalogs | Inbound Product Sharing from the navigation bar.
This displays the Inbound Product Sharing manager.

-
Select Activate all shared products.
-
Click Apply to save your settings.
This assigns all shared master products to the channel repository.
