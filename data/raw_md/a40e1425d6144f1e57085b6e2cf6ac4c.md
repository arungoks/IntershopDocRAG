# Application Management

Intershop Commerce Management can run multiple applications in the context of a channel or site. Channel managers use the Applications Manager to create applications, to enable or disable applications as well as to set application-specific preferences.

The present page serves as an overview for the topic. Available tasks for this topic can be found here:
