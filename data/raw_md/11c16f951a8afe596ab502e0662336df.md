# Editing Attribute Mapping Rules for Product Syndication

-
Select the intended channel from the context selection box, then select Mass Data Tasks | Product Syndication.
This displays the Syndication & Synchronization overview page.
-
Select Mapping Rules.
This displays a list of all source repositories from which products in the current repository have been derived.
-
Select the Edit Rules link for the source repository for which you want to edit mapping rules.
This opens the Attribute Mapping Rules dialog. Edit settings as required and click Apply. For an overview of available rules, see
[Attribute Mapping Rules](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/concept_product_syndication.html#con_mdata_sdc_mapping).
