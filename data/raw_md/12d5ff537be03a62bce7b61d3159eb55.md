# Sorting Payment Method Configurations

You can define the order in which the payment methods are listed in the storefront applications. To sort payment methods:
-
Select the channel from the context selection box, then select Orders | Payment Methods.
This displays a list view of all payment methods of the sales channel.

-
Select the checkboxes of the payment methods you intend to move up or move down in the list.
-
Use the arrow buttons in the sorting column to change the payment method order.
Clicking an arrow button moves the corresponding payment method up or down, immediately changing the display order in the storefront.
