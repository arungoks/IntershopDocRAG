The [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management) rule engine, which is based on [Drools](http://www.drools.org/), provides an abstract way of executing rules on [business object](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2158101810/business+object)s (like [promotion](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2153813271/promotion)s, [shipping](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214729962/shipping) conditions, etc.). [User](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2160822248/user)s or developers can create, load, store, and remove rules, as well as manage rules and execute them without the need to know any specific rule engine language as long as they use the interfaces provided by the rule engine implementation. In addition, the rule engine serves a [framework](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2163970267/framework) that allows for integrating another rule engine.
