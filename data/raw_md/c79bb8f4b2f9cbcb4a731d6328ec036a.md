# Managing Search Index Keywords

Note:

For this feature to work, your Intershop Commerce Management system must have integrated a third-party search engine, see

[Search Index Services](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/concept_search_index_services.html).To manage search index keywords:

-
Locate the product for which you want to manage search engine attributes.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product.
The General tab of the Standard Product dialog is displayed.
- Change to the Attributes tab.
-
In the Search Index Keywords section, specify the intended keywords.
Separate new entries using a line break.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
-
Rebuild the search index.
For details, see
[Managing Search Indexes](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_search_indexes.html).Note:Make sure that "Search Index Keywords" is included as an indexed attribute.
