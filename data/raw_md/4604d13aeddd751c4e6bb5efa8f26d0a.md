# Viewing App Details

You can access the details of an app from the user detail page. To view app details:

-
Navigate to the user detail page.
For guidance, see
[Viewing User Details](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_viewing_user_details.html). -
On the user detail page, click the
*More*icon (…) of the app in question (for example,*Admin Center*).A drop-down menu is opens. -
Click View details in the drop-down menu.
The detail page of the selected app opens.
