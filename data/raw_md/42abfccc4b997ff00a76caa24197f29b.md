Functions are used in [ISML Expressions](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775688409/Reference+-+ISML+Expressions) to perform a specific task and return a specific value when the template is processed to generate its output. ISML functions are, for example, used to dynamically generate URLs for links to other storefront pages (`url()`

, `urlex()`

), or to operate on strings.

ISML functions can be categorized into the following groups:

[url()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775690000/ISML+Function+-+url), [urlex()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775690011/ISML+Function+-+urlex), [ISML Function - action()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775687382/ISML+Function+-+action), [ISML Function - parameter()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775689514/ISML+Function+-+parameter), [webRoot()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775690028/ISML+Function+-+WebRoot), [webRootEx()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775690046/ISML+Function+-+WebRootEx), [ISML Function - ContentURL()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775688168/ISML+Function+-+ContentURL), [ISML Function - ContentURLEx()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775688251/ISML+Function+-+ContentURLEx), [ISML Function - sessionlessurl()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775689606/ISML+Function+-+sessionlessurl), [ISML Function - sessionlessurlex()](https://knowledge.intershop.com/kb/go.php/a/ENFDEVDOC/pages/1775689609/ISML+Function+-+sessionlessurlex)
