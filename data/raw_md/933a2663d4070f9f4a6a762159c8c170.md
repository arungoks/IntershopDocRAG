In [Intershop Commerce Management](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2217982072/Intershop+Commerce+Management), an extension point is a place within a [code artifact](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2214730413/development+artifact) where other code artifacts can be plugged in in order to extend the original functionality. Currently, there are three types available for the particular code artifacts: Java, ISML template and pipeline.
