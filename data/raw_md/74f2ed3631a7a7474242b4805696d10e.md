# Deleting Catalog View

To delete a catalog view:

-
Select the channel from the context selection box, then select Catalogs | Catalog Views from the navigation bar.
This displays a list of existing catalog views.
-
Select the catalog view to be deleted.
Check the box in the row of the catalog view you intend to delete.
-
Click Delete, then OK to confirm the deletion.
The catalog view is removed from the system.
