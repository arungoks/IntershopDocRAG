# Exporting Classification Catalogs

To export catalogs in XML format:
-
In the navigation bar, select Import & Export.
The overview page is displayed.

-
In the Classification Categories (XML) row, click Export.
A list of available catalogs is displayed.

-
Select the catalog to export and provide a name for the export file.
-
Click Export.
The export process is started immediately. The Import & Export Status dialog is displayed.

-
Download the export file.
