Discover the cashback promotion feature of Intershop Commerce Management.

The cashback feature enables customers to receive an adjustable amount of cashback money as a coupon code after their purchase. In this way customers have an incentive to make further purchases.

This video tutorial shows how a simple cashback promotion can be set up.


This video tutorial shows all relevant steps to create a basic cashback promotion. This includes the following tasks:

Furthermore, the video tutorial shows how a customer can earn and redeem cashback in the storefront.

Click *New*.

Provide the following information:

| Name | Cashback Promotion Code Group |
|---|---|
| Active | |
| ID | cashback |
| Description | Cashback promotion codes will be stored in this promotion code group. |

Click *Apply*.

Enter the following information:

| Value | Additional Information | |
|---|---|---|
| Name | Cashback Earning | |
| Enabled | ||
| ID | cashback_earning | |
| Specific Priority | 0 | Use the lowest priority (0) to ensure that the cashback earning is the last promotion and all other discounts are subtracted first. |
| Calculation | Discounted Price | The calculation should be based on the discounted price to ensure that the cashback is earned for the final price where other discounts are subtracted first. |
| Combination | Freely Combinable | Cashback earning promotions are always combinable with any other promotion, independent of this setting. |

Enter the following information:

| Value | Additional Info | |
|---|---|---|
| Name | Cashback Redemption | |
| Enabled | ||
| ID | cashback_redemption | |
| Specific Priority | Default Priority 1.100 | The cashback redemption promotion has the highest priority per default. |
| Combination | not combinable | Select whether this promotion should be combinable or not. Due to the fact that it has the highest default priority, this promotion will always be the first (and only one, if not combinable) that is executed. |

Switch to the *Target Group* tab and set *Customer Segments - Reigistered Customers *as target group.

Info

Since the cashback is stored customer-specifically, the customer check is executed implicitly during the code redemption process. Therefore the target group can be set to *Everyone* as well.

Tip

The promotion code search allows to retrieve status information for individual promotion codes.

This can either be done with the cashback promotion code search, which allows to search by customer name or e-mail or with the regular promotion code search which allows to search by the coupon code itself.
