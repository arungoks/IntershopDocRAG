# Removing Users from Modules

To remove users from a module:

-
In
*Apps & modules*, click on the module of an app, for example User management. For details, see[Viewing Apps and Modules](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/apps_modules/task_viewing_the_apps_and_modules_overview.html).The detail page of the selected module opens. -
Click the
*Remove user*icon () of the user in question. -
Click the Remove button to confirm the removal of the user (or discard this process by clicking the Close button).
The user is removed from the list. The number of available seats is increased.
