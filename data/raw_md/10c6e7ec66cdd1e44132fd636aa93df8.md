This migration guide describes changes related to the location of the definition file for global placeholders used in customer specific assortments.

The location of the global placeholder definition file *searchquerydefinition-parameter-bindings.xml* has been changed from the *share/system/config/cluster* directory to the *sites/root/1* directory.
