# Managing Personal Data Request Preferences

Intershop Commerce Management allows to manage preferences for personal data requests for a channel. To do so:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select Personal Data Request Preferences.
The detail page is displayed. The following options are available:
Table 1. Personal Data Request Preferences Preference Description Link Expiration Period The number of hours after which the confirmation links in personal data request confirmation e-mails will no longer be valid. The e-mail is triggered when a personal data request is created in Intershop Commerce Management. Deletion Period The number of days after which personal data requests which are in a closed or pending state will be deleted automatically. E-mail From Address Defines the sender's e-mail address. -
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
