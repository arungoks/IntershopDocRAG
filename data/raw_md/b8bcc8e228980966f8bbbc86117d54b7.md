Generally, a software framework provides the possibility to extend a software with new [feature](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152516365/feature)s with the minimum possible development effort. From a developer's perspective, a framework is code intended to be specialized and/or instantiated in order to implement specific [feature](https://knowledge.intershop.com/kb/go.php/a/ISGLOS/pages/2152516365/feature)s.
