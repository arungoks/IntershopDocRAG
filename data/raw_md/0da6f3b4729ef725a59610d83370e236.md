# Downloading Order Documents

- Invoices
- Aggregated invoices
- Credit notes
- Delivery notes
- Return labels
- Return slips

-
Search for the order.
For details, see
[Searching Orders](https://knowledge.intershop.com/oh/go.php/iom/master/iom_en/topics/managing_orders/topic_searching_orders.html#task_order_search). -
In the result list, click on the order number you want to edit.
This opens the order detail page.
-
Click Documents.
This displays the available order documents.
-
Click the Download
icon next to
the document in question.
This prints the document into a PDF file and prompts you to download it.
