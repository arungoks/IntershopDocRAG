# Managing Content Assets

Depending on an organization's business model and, consequently, the implementation of the storefront application(s), content managers and content editors can access and edit content assets on three levels: the organization, the channel or the application. Using the context selection box (as described in [User Interface Overview](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/introduction/topic_user_interface_overview.html#ov_ui)), content managers and content editors select the intended context (or level), and then access the content assets via the Intershop Commerce Management tools introduced below.
