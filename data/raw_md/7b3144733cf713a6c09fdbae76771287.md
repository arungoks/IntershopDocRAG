# Publishing Components

[Data Replication Task Management](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_data_replication_task_management.html)).

Only if content sharing (see [Content Asset Sharing](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/content_management/topic_content_asset_sharing.html)) is disabled, components can also be published manually as follows:

-
Select the management context from the context selection box, then select Content | Components.
This opens the Components list view.
-
In the list view, select the checkboxes of the components to be published.
Alternatively, perform a search and click Select All to select the search result.
-
Click Publish to Live System.
This displays the list of configured target systems.
-
Select the intended target system and the locking conflict resolution.
For details about the locking conflict resolution options, see
[Object Locking](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/concept_catalog_import_and_export.html#con_cat_impex_lock). -
Click Finish.
The selected components are replicated to the selected target system.
