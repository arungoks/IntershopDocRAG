# Content Management: Concepts

Intershop Commerce Management provides a powerful and intuitive authoring mechanism that allows catalog managers and content managers to easily edit and manage storefront pages.

The content management is based on a set of application type-specific models and their implementations. Generally, they enable developers to model "pagelets", i.e., the page and component building blocks, for a storefront application, and users to create and edit the corresponding page and component instances in Intershop Commerce Management.
