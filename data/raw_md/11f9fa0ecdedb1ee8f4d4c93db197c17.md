# Exporting Orders

Note:

To export orders:
Currently, the Order Management Tool supports CSV file export only.

-
Search for the orders.
For details, see
[Searching Orders](https://knowledge.intershop.com/oh/go.php/iom/master/iom_en/topics/managing_orders/topic_searching_orders.html#task_order_search). -
On the top of the result list, click the icon and select CSV download.
This immediately exports the result set into a CSV file and prompts you to download it.
