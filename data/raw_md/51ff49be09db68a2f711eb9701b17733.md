# Managing Image Type-View Associations

Managing image type-view associations for products involves adding, editing and removing image type-view associations.

## Adding Image Type-View Associations

-
Locate the product you want to edit.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product you intend to edit.
The General tab of the Standard Product dialog is displayed.
-
Lock the product.
For details, see
[Locking Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locking_products.html#task_pim_mgmt_lock). -
Select the intended locale and click Apply.
This displays the attribute values for the selected locale.
-
In the Additional Images section, select the intended image view.
This enables the Image Type drop-down list.
-
Select the intended image type.
This enables the Image field.
-
Specify the path to the intended image file or click the File Select button to browse for the intended image file.
The Select Image dialog allows for uploading new image files, similar to the behavior described in
[Uploading Product Images](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/topic_static_file_importexport.html#task_pim_impex_static_uplImage).Note:When using an image type with an empty prefix, there is an additional drop down box that lets you chose the location in the shared file system, where the images to be assigned come from (channel/organization). However, this selection is only available for channels, not in the organization. For details, refer to[Creating Image Types](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/topic_managing_image_types.html#task_pim_images_types_create). -
Click Add.
This adds the selected image type-view association to the current product.

## Editing Image Type-View Associations

To edit the images of individual image type-view associations:

-
Locate the product you want to edit.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product you intend to edit.
The General tab of the Standard Product dialog is displayed.
-
Lock the product.
For details, see
[Locking Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locking_products.html#task_pim_mgmt_lock). -
Select the intended locale and click Apply.
This displays the attribute values for the selected locale.
-
In the row of the image type-view association in the Additional Images section, either specify the path to the intended image file or click the File Select button to browse for the intended image file.
The Select Image dialog allows for uploading new image files, similar to the behavior described in
[Uploading Product Images](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/topic_static_file_importexport.html#task_pim_impex_static_uplImage). -
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.

## Deleting Image Type-View Associations

To delete individual image type-view associations:

-
Locate the product you want to edit.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product you intend to edit.
The General tab of the Standard Product dialog is displayed.
-
Lock the product.
For details, see
[Locking Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locking_products.html#task_pim_mgmt_lock). -
Select the intended locale and click Apply.
This displays the attribute values for the selected locale.
- In the Additional Images section, select the image type-view association(s) you intend to delete.
-
Click Delete, and confirm the operation.
The selected image type-view associations are removed from the current product.
