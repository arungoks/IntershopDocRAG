# Creating Connections

This section describes how to establish a new connection. Supported connection types are ICM (Intershop Commerce Management), PWA (Intershop Progressive Web App), and other services. To create a new connection:

-
Click Connection management in the main content area on the welcome screen, or in the sidebar.
The
*Connection management*overview page opens. Here you can see a list of all connections. -
Click Create connection and select the type of connection you want to create from the drop-down menu.
Alternatively, click + Create for the desired connection in the list.Note:Currently only one instance of Intershop Commerce Management can be connected.
-
Specify all required data (
*ID*,*Type*,*URL*) and relevant custom attributes.You can use the Test button to verify that the provided URL is accessible. For information on the configuration parameters, see[Custom Attributes](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/connection_management/concept_custom_attributes.html). -
Click Create.
This creates the connection based on the specified data.
