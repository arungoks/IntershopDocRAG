# Managing Administration Users

This section discusses how to create and maintain system administration users.

Intershop Organization Management allows for creating and maintaining individual system administration users. Intershop Commerce Management supports various types of system administration users, which are represented using dedicated access privileges:

| Access Privilege | Description |
|---|---|
| Administrator | Responsible for business administration tasks to be performed in Intershop Organization Management, such as managing enterprise organizations, classification systems, application settings and controlling data replication. |
| Catalog Manager | Responsible for creating and managing catalogs. |
| Data Replication Manager | Responsible for data replication management. |
| SMC Administrator | Responsible for system administration tasks in an Intershop Commerce Management cluster to be performed in Intershop System Management. |
| SMC File Administrator | Responsible for file administration in Intershop System Management. This allows to view and download files from any server within an Intershop Commerce Management cluster. |
| SMC Installation Administrator | Responsible for installation maintenance in Intershop System Management. This includes the generation of reports and dump files. |
| SMC License Administrator | Responsible for creating license reports in Intershop System Management. |
| SMC Log Administrator | Responsible for configuring logging in Intershop System Management. This includes customizing existing logging appenders and uploading configuration files. |
| SMC Monitor Administrator | Responsible for monitor administration in Intershop System Management. This includes monitoring properties, process information, current threads and requests of all server components. |
| SMC Schedule Administrator | Responsible for schedule administration in Intershop System Management. This includes creating, deleting and editing scheduled jobs as well as managing process chains. |
| SMC Site Administrator | Responsible for site administration in Intershop System Management. This includes managing general site preferences, page cache preferences and application settings. |
| Transport Manager | Responsible for creating, editing and deleting file transport process configurations. These allow to transfer files from one netwok location to another. |
| User Manager | Responsible for creating, editing and deleting operational users. |
