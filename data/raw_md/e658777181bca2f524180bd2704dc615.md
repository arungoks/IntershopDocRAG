# Listing Sales Organizations

To view a list of all sales organizations in the system:
In the navigation bar, select Sales Organizations.
A list of all sales organizations in the system is displayed.

Note: If you have initialized the database without importing the demo data in a new Intershop Commerce Management instance, this list should be empty.
