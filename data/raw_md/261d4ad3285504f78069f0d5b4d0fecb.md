# Removing Apps or Modules from a User

*User management*:

-
Navigate to the user detail page, see
[Viewing User Details](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_viewing_user_details.html). -
Either remove an app or a module:
-
Click the Remove button (or discard this process by clicking the Close button).
The Remove dialog closes and a confirmation message appears.Note:You cannot remove the
*Admin Center*app and its modules from any admin user or from yourself. The Remove app option is not visible when these users have been selected. Furthermore, this feature is not available if you have no permission for Application management.
