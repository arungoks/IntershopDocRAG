Welcome to the IOM dbaccount Docker image.

This image is used to create and configure the PostgreSQL user and database, suitable for use with IOM.

The IOM dbaccount image provides backward compatibility to many IOM versions. For the best experience, please always use the newest version of the IOM dbaccount image, according to the compatibility matrix.
