# Creating Products

-
Select the intended management context from the context selection box, then select Catalogs | Products.
This displays the product list view.

-
Click New.
The General tab of the Standard Product dialog is displayed.

-
In the General tab, specify the details as necessary.
Mandatory data include the name and the ID.

-
Click Apply.
The new product is created and the additional tabs become active.

-
Under each tab, add the appropriate information about the new product.
For details about product editing options, see the sections below.
