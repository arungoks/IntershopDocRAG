# URL Rewriting

URL rewriting is a mechanism that enables Intershop Commerce Management sites to publish search engine-friendly URLs. Shop managers can enable or disable this functionality on channel level.

Generally, Intershop Commerce Management can define rules that hide or add specific parts of the visible URLs. The intention is to provide qualified URLs, which are short and readable and contain enough semantic information to describe the content of the addresses. Intershop Commerce Management ships with an initial set of URL rewriting rules. Contact your system administrator if you require new rules or modifications on a project-specific base.

For details about enabling URL rewriting, see [Enabling URL Rewriting](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_seo_settings.html#task_pref_gen_seo_url).
