# Managing Retail Sets

Retail sets combine two or more products into a "product group", which is intended to be presented together. To define a retail set:

-
Locate or create the product to represent the retail set.
For details, see
[Locating Products](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/catalogs_products/task_locating_products.html). -
Click the name of the product.
The General tab of the Standard Product dialog is displayed.
-
Change to the Retail Sets tab.
This displays two lists: a list showing additional products bundled with the current product into a retail set, and a list showing all retail sets which include the current product.
-
In the Products tab, click New.
The product list view is displayed.
-
Locate the products to be included with the retail set, select the respective check box for each product, and click OK.
The products included in the retail set are now displayed in a table.Note:Retail sets shared from the master repository can be edited only in the master repository.
