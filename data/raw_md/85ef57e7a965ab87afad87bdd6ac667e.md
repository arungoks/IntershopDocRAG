# Navigation

The Intershop Customer Engagement Center user interface is divided into four major areas: the header, the contact session bar, the main content area and the footer.

-
*Header*The header section provides several key functionalities:

-
*Contact session bar*The contact session bar provides control of the contact session. It displays general customer information and provides a button to end the contact session, irrespective of the view that is displayed in the main content area.

-
*Main content area*The main content area corresponds to the context of an individual contact session. In a tabbed dialog, you can edit customer data, edit the customer's carts, place orders, start a storefront impersonation session and generate/apply coupons. If there is no contact session running, you are presented with the customer search page.

*Footer*The footer displays version information of the app and the name of the tenant through which you are currently logged in.
