# Deleting a Process Chain

-
In the navigation bar, select Mass Data Tasks | Process Chains.
The Process Chains list is displayed.

-
Mark the process chain's checkbox.
Please note that you may select multiple process chains at once.

-
Click Delete.
Click OK to confirm.
