# Installation Maintenance

This section describes the installation maintenance features of Intershop System Management including the generation of reports and dump files.

Using the Intershop System Management Installation Maintenance module, the system administrator can collect vital information about the Intershop Commerce Management system and transfer it to Intershop Support.
