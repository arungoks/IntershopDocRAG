# Viewing Order Payment Details

To view the payment details of an order:
-
Locate the order you want to view.
-
Click the ID of the intended order.
This opens the Order Detail view.

-
Change to the Payment tab.
This displays all payment transactions of this order. For each payment transaction, the table shows the payment amount and the current payment status. To view the history of a payment transaction, click

History in the corresponding row. This displays the transaction history and, if implemented, allows for payment processing operations as supported by the current payment service (see also

[Generic Payment Processing](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/concept_generic_payment_processing.html)).
