# Assigning Users to Apps

You can assign existing users to an app. Existing means that the user has already been added to the Intershop Administration Portal and can be found in the user list. For details, see [Adding Users](https://knowledge.intershop.com/oh/go.php/iap/master/iac_en/iac/user_management/task_adding_users.html#task_inviting_users).

To assign users to an app:

-
In
*Apps & modules*, click on an app (for example, Admin Center).The detail page of the selected app opens. -
Click + Add (or + Add user in case no users have been assigned to this app yet).
The corresponding dialog with a user search opens.
- Select one or more users from the list or enter the username or e-mail address into the search input field.
-
Click the Add button (or discard this process by clicking the Close button).
The dialog closes and a confirmation message appears. The number of available seats is decreased. The added user receives an notification e-mail.Note:It is possible to add more users than available seats, but they do not have access to the app.
