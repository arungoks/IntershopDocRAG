# Assigning Products by Search

-
Select the channel from the context selection box, then select Catalogs | Product Data Feeds from the navigation bar.
A list of all configurations currently defined is displayed.
- Click the name of the configuration to which you want to assign products.
-
Change to the Products tab.
A list of all products currently assigned to the configuration is displayed.
-
Click Assign by Search.
The product list view is displayed.
- Select the corresponding check boxes in front of the products to be assigned.
- Click Assign Selected.

Note:

Newly created products will need to be assigned manually to the product data feed when using Assign by Search.
