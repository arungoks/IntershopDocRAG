# Deleting Product Data Feeds

To delete one or multiple product data feeds:
-
Select the channel from the context selection box, then select Catalogs | Product Data Feeds from the navigation bar.
A list of all data feeds currently defined is displayed.

-
Select the product data feed(s) which should be deleted.
-
Click Delete, then OK to confirm your action.
The product data feed(s) is/are deleted from the system.
