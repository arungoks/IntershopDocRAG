# Deleting Short Links

-
Open the Short Links manager.
-
Select the channel from the context selection box, then select Marketing | Link Management.
-
Click Short Links.

This displays the list of existing short links.

-
Select the check box for the short link to be deleted.
-
Click Delete, then OK to confirm the deletion.

Short links can also be deleted using the Delete button in the details view.
