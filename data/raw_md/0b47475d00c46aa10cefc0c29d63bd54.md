# Changing Administration User Password

-
In the navigation bar, select Users.
A list of all system administration users is displayed.
-
Click the name or ID of the user you intend to edit.
This displays the user detail view.
- Ensure that you have specified a valid e-mail address, as you need access to it to change the password.
-
Click New Password.
The system immediately generates a single-use password and sends it to the specified e-mail address. The next time the user logs in, the system immediately sets this password to "expired" and prompts the user to specify a new password.

Warning:

Intershop strongly recommends to either change the default administration user password at the first login, or to create new administration users with dedicated access rights and to delete the default administration user.
