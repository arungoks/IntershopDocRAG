# Deleting Administration Users

To delete one or more system administration users:
-
In the navigation bar, select Users.
A list of all system administration users is displayed.

-
Select the checkboxes of the users to be deleted.
-
Click Delete, and confirm the operation.
The selected users are removed from the system.


Note: Objects are completely deleted (and the ID made available again) after the next `DeleteDomainReferences`

job has run.
