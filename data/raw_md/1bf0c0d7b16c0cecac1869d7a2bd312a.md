# Managing Recurring Order Preferences

Note:

This feature is provided through the

*Recurring Order Extension*and is not available out of the box. For details, refer to[Overview - Recurring Order Extension](https://knowledge.intershop.com/kb/index.php/Display/31Q320)in the Intershop Knowledge Base. To learn more about recurring orders in general, refer to[Recurring Order](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/concept_recurring_order.html).Channel administrators can enable or disable recurring orders. To do so:

-
Select the management context from the context selection box, then select Preferences.
This displays the Preferences overview page.
-
Select Recurring Orders.
The details page is displayed.
-
Select the checkbox Recurring Orders.
To disable recurring orders, deselect the checkbox.Warning:If you disable recurring orders while there are existing recurring orders, the UI will hide all related sections, and no more recurring orders will be created. Affected customers will not be notified.
-
Click Apply to save your settings.
Otherwise, click Reset to discard your changes.
