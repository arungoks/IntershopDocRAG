# Store Management and Store Finder

Using Intershop Commerce Management, organizations can manage subsidiary "physical" stores. This feature supports the special shipping method "Pick Up in Store", intended to serve models in which orders are fulfilled via subsidiary stores.

If customers chose the in-store pick-up, they can select their store upon checkout using a store finder wizard. The store finder wizard as shipped with the reference storefront exposes various data entry fields to narrow down the range of stores, including country, ZIP/postal code, city, and state/province.

The stores offered through the store finder wizard are created and managed by the channel order manager in Intershop Commerce Management. In Intershop Commerce Management, stores can be created manually or imported from an XML data source.

For details about store manangement, see [Store Management](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_store_management.html#task_store_mgmt).

Enabling the in-store pick-up as a shipping method involves:

-
Activating the corresponding general preference, see

[Enabling Pick Up in Store](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/operation_maintenance/topic_managing_shipping_settings.html#task_pref_gen_ship_pickup) -
Defining an actual "Pick Up in Store" shipping method, see

[Managing Shipping Methods](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/topic_managing_shipping_methods.html#task_order_ship_method) -
Defining a corresponding destination region to group the addresses of the type "store", see

[Managing Destination Regions](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/topic_managing_destination_regions.html#task_order_ship_region) -
Combining the "Pick Up in Store" shipping method and the corresponding destination region with a freight class, which is then to be associated with products or catalog categories, see

[Managing Freight Classes](https://knowledge.intershop.com/oh/go.php/icm/master/icm_main_en/orders_fulfillment/topic_managing_freight_classes.html#task_order_ship_freight).
